# TopFeed 
국내 유명 커뮤니티의 인기 글을 한번에 볼 수 있는 웹사이트입니다.<br>
이메일을 등록하시면 이메일로 간단한 게시글들을 받아보실 수 있습니다.<br>
위의 로고를 클릭하면 서비스 이용이 가능합니다.
